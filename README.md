# 🎮 Kho Tài Nguyên Game Tuổi Thơ (Instant Retro Downloader)

Chào mừng bạn đến với dự án website **Tổng hợp Game Tuổi Thơ**. Đây là nền tảng tối ưu giúp bạn sở hữu trọn bộ công cụ chơi game retro chỉ với **một cú click chuột**. 

Website được lập trình để tự động hóa quy trình tìm kiếm và tải tài nguyên, giúp bạn tiết kiệm thời gian tối đa.

## 🚀 Cơ chế hoạt động (One-Click Download)
Điểm đặc biệt của website này là hệ thống **điều hướng tự động**:
- **Nhấp vào Biểu tượng/Tên máy:** Hệ thống sẽ tự động đưa bạn đến link tải các bộ **ROM Game** tương ứng của hệ máy đó.
- **Nhấp vào Link Giả lập:** Website sẽ tự động kích hoạt lệnh tải xuống hoặc dẫn trực tiếp đến trang tải file của các trình giả lập uy tín nhất.

## ✨ Các tính năng chính
- **Smart Redirect:** Tự động nhận diện và chuyển hướng người dùng đến đúng kho tài nguyên cần thiết.
- **Tải xuống trực tiếp:** Hỗ trợ tải các file giả lập (.exe, .apk) và file game (.nes, .gba, .iso...) một cách nhanh chóng.
- **Phân loại rõ ràng:** Hệ thống biểu tượng trực quan cho từng hệ máy (NES, GBA, NDS, PSP...).
- **Giao diện thân thiện:** Thiết kế tập trung vào sự đơn giản, nhấn vào là tải, không cần thao tác rườm rà.

## 🛠️ Hướng dẫn sử dụng
1. **Tìm máy:** Nhấn vào biểu tượng hệ máy bạn muốn chơi để nhận link tải ROM.
2. **Tải phần mềm:** Nhấn vào mục giả lập để hệ thống tự động tải file chạy về máy.
3. **Hoàn tất:** Mở trình giả lập và nạp ROM vừa tải để bắt đầu hành trình tuổi thơ.
